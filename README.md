# auth-example

### Clone using ssh:
```sh
$ git clone git@github.com:andersonscherrrer/auth-example.git
```
### Clone using https (require password):
```sh
$ git clone https://github.com/andersonscherrrer/auth-example.git
```
### Running api-node
```sh
$ cd auth-example/api-node
$ npm install
$ node server
```
